Just clone this project using:

```cmd
git clone <repo url>
```
After use cd:
```cmd
cd travel-booking
```

And finally build the project by:
```cmd
docker-compose up --build
```

Use url like to get documentation:
```
http://localhost:9091/swagger-ui/index.html#/
http://localhost:9092/swagger-ui/index.html#/
```
